# Portfolio 8-Bit

Welcome to the Portfolio 8-Bit project! This project is a personal portfolio webpage designed with an 8-bit theme inspired by classic Game Boy aesthetics. It showcases various sections that highlight the portfolio owner's skills, experiences, and projects.

## Project Structure

The project is organized as follows:

```
portfolio-8bit
├── src
│   ├── components
│   │   ├── Home.tsx         # Landing section with a welcome message
│   │   ├── About.tsx        # Information about the portfolio owner
│   │   ├── Experience.tsx    # List of professional experiences
│   │   ├── Projects.tsx      # Showcase of completed projects
│   │   └── Contact.tsx       # Contact form or information
│   ├── styles
│   │   ├── global.css        # Global styles and resets
│   │   ├── gameboy-theme.css # 8-bit Game Boy themed styles
│   │   └── components.css     # Styles specific to individual components
│   ├── assets
│   │   └── fonts
│   │       └── press-start-2p.ttf # Custom font mimicking Game Boy text style
│   ├── App.tsx               # Main application component
│   └── index.tsx             # Entry point of the application
├── public
│   └── index.html            # Main HTML template for the React application
├── package.json              # npm configuration file
├── tsconfig.json             # TypeScript configuration file
└── README.md                 # Project documentation
```

## Getting Started

To get started with the Portfolio 8-Bit project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd portfolio-8bit
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to view the portfolio.

## Components Overview

- **Home**: The landing section that welcomes visitors and provides navigation links to other sections.
- **About**: A section that details the portfolio owner's background, skills, and interests.
- **Experience**: A comprehensive list of professional experiences and relevant positions held.
- **Projects**: A showcase of various projects, including descriptions and links to live demos or repositories.
- **Contact**: A section that provides a contact form or contact information for potential clients or employers.
